# OIBSIP-Javadevelopment-Task4
The Online Examination System is a Java-based console application developed using JDBC and MySQL. It provides a simple platform for conducting multiple-choice examinations online.The system allows users to log in securely, update their profile and password, attend exams, answer MCQ-based questions, and receive scores automatically after submission.

Features
User Login Authentication
Update Profile and Password
Multiple Choice Questions (MCQs)
Exam Timer with Auto-Submit Functionality
Automatic Score Calculation
Result Storage in MySQL Database
Logout and Session Management
JDBC Connectivity with MySQL
Technologies Used
Java
JDBC
MySQL
VS Code
MySQL Workbench
Database Tables
users
questions
results
Project Workflow
User logs in using valid credentials.
User can update profile details or password.
User starts the examination.
Questions are fetched from the MySQL database.
User selects answers for MCQs.
Timer monitors the exam duration.
Exam is auto-submitted when time expires.
Score is calculated and stored in the database.
User can view results and logout securely.

This project demonstrates Java database connectivity, CRUD operations, authentication, session management, and timer-based examination functionality. It is suitable for beginners learning Java, JDBC, and MySQL integration.
